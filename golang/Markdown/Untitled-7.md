# Markdown 表格
## 语法格式
| 表头 | 表头 |
| --- | --- |
| 单元格 | 单元格 |
|单元格  | 单元格 |

## 对齐方式
>---: 设置内容和标题栏居右对齐。
>:--- 设置内容和标题栏居左对齐。
>:---: 设置内容和标题栏居中对齐。

| 左对齐1 | 右对齐1 | 居中对齐 |
| :-----| ----: | :----: |
| 单元格 | 单元格 | 单元格 |
| 单元格 | 单元格 | 单元格 |

## 单元格复杂化处理
>表格单元格内可以使用大部分的Markdown语法：

| 功能 | 描述 | 状态 |
|------|------|:----:|
| **用户登录** | 支持邮箱和手机号登录 | &#x2705; |
| *密码重置* | 通过邮箱重置密码 | &#x26a0;&#xfe0f; |
| `API接口` | RESTful API 设计 | &#x2705; |
| [文档链接](https://example.com) | 查看详细文档 | &#x1f4d6; |

- 换行处理  
在需要换行的地方加上`<br>`
