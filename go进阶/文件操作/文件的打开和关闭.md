# 常用的文件操作函数和方法

1. 打开一个文件进行读操作：

   ```go
   os.Open(name string)(*File,error)
   ```

2. 关闭一个文件:

   ```go
   func (f *File) Close() error
   ```

3. 案例演示:

   ```go
   package main
   import (
   	"fmt"
   	"os"
   )
   func main() {
   	//打开文件
   	//file 1,file对象 2,file指针 3,文件句柄
   	file, err := os.Open("e:/test.txt")
   	if err != nil {
   		fmt.Println("open file err=", err)
   	}
   	//输出的是个地址
   	fmt.Println("file=%v", file)
   	//关闭文件
   	err = file.Close()
   	if err != nil {
   		fmt.Println("close file err=", err)
   	}
   }
   
   ```
